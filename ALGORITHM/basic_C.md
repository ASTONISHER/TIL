# 문자열, 문자
<h3>1. 문자열과 문자</h3>

```C
    char arr1[] = {'H', 'i', '!'};
    // 배열의 크기는 3
    
    char arr2[] = "Hi!";
    // 배열의 크기는 4, 마지막에 널문자 삽입되어있음
``` 
- 문자열은 끝을 확인하기 위해서 마지막에 널(\0)문자가 자동으로 삽입된다. 
- printf함수로 출력할 때에도 널 문자로 문자열을 구분해 냄.
- 널문자의 ASCII값은 0이므로 아래 두 코드는 같은 의미
```C
    str[1] = '\0';
    str[1] = 0;
```
- 문자열의 길이 계산 코드
```C
    while(arr[len] != '\0')
      len++;
```
